# Zak's Barndominium Budget Calculator

WordPress plugin that adds a customer-facing barndominium budget calculator.

## Installation

1. Upload `barndominium-budget-calculator.zip` in WordPress under Plugins > Add New > Upload Plugin.
2. Activate the plugin.
3. Add this shortcode to any page:

```text
[barndominium_budget_calculator]
```

Short alias:

```text
[barndo_budget_calculator]
```

Optional contact URL override:

```text
[barndominium_budget_calculator contact_url="https://zaksbuilding.com/contact/"]
```
